# 资源文件说明

## Tab Bar 图标

需要以下 81x81 像素 PNG 图标文件，放在 `assets/icons/` 目录：

| 文件 | 用途 |
|------|------|
| home.png | 首页（未选中） |
| home-active.png | 首页（选中，绿色 #07C160） |
| browse.png | 分类（未选中） |
| browse-active.png | 分类（选中） |
| search.png | 搜索（未选中） |
| search-active.png | 搜索（选中） |
| mine.png | 我的（未选中） |
| mine-active.png | 我的（选中） |

## 默认封面图

- `assets/default-cover.png` — 菜谱默认封面图（当没有上传图片时显示）

> 提示：可以使用 iconfont 或 sketch 导出 tab bar 图标。建议找设计师出图或使用免费图标库。
